Intro

Toyin L. Ola, MA, MPH, MSW is a public health data analytics professional with experience wrangling, aggregating, and visualizing tabular, survey, and geospatial data. To date, her analytics work has involved streamlining quarterly reporting of survey results using the Qualtrics platform's API and R Markdown, generating static and interactive geospatial visualizations using R, developing dashboards in Excel and in Tableau, and more. At work and in her free time, Toyin enjoys honing her technical skills and learning from fellow open source enthusiasts: She is currently on a mission to learn natural language processing (NLP) techniques and JavaScript, especially D3. One of her greatest technical achievements is navigating the complexities of tidy evaluation and quasiquotation while writing user-defined functions to make iteration more efficient. See Toyin's work and educational history on LinkedIn. See sample projects here.

Projects

Starbucks Shiny Web Application | This project is currently in progress.

Static Data Viz Gallery | This is a small collection of non-interative data visualizations created using {sf}, {ggplot2}, and ggplot extension packages. Skills gleaned from R for the Rest of Us' Mapping with R course were also applied to create a few choropleth maps.

Bakeries Map | This sample project showcases web scaping, geocoding, and mapping. The result is a flexdashboard featuring an interactive {leaflet} map of bakeries in the Richmond, Virginia metropolitan area. A searchable reactable is included for those who want to access the data in that format. The {crosstalk} package is used to enable interactivity while maintaining a self-contained product that can be readily shared without needing a server or a live R session.

About

Toyin (she/her) is passionate about evaluating health programs and delivery systems with the aim of making them more efficient and effective for their intended beneficiaries. She strives to understand and elucidate the social and behavioral determinants of health and to help individuals navigate life with chronic illness.
